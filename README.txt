# Library System (Python Based)

## Introduction :

This is a simple command-line-based library management system written in Python. It allows users to perform various operations like creating an account, searching for books, adding books, and loaning books.

## Features :

- **Create Account**: New users can create an account by providing their personal information.
- **Log In**: Existing users can log in using their credentials.
- **Search for Books**: Users can search for books by title, author, publisher, or publication date.
- **Add a Book**: Librarians can add new books to the library's collection, specifying book details.
- **Borrow a Book**: Users can borrow available books. The system keeps track of borrowed books and available copies.
- **Return a Borrowed Book**: Users can return books they have borrowed. The system updates the available copies.
- **See Available Copies of a Book**: Users can check the number of available copies of a specific book.
- **Modify User Details**: Users can edit their personal information, such as name, address, email, and date of birth.
- **Modify Book Details**: Librarians can update book information, including title, author, year, and publisher.
- **Remove a Book**: Librarians can remove books from the library's collection.
- **Show Borrowed Books**: Users can view the list of books they have borrowed.
- **Log Out**: Users can log out of their accounts.
- **Exit**: Users can exit the application.


## Usage :

1. Run the Python script (`app.py`) in your terminal.
2. Follow the on-screen menu prompts to navigate through the system.
3. If you are a new user, choose "Create Account" to set up your account.NOTE(Demo Users have been created to make it easier to use)
4. If you are an existing user, select "Log In" to access your account.
5. Use the menu options to search for books, borrow and return books, and manage your account information.
6. Librarians can add, modify, and remove books from the library's collection.
7. The system keeps track of borrowed books and available copies.
8. You can log out or exit the application when finished.

## Dependencies
This script does not have external dependencies and can be run using Python 3x.x.
