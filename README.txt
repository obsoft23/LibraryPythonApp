# Library Management System

## Introduction
This is a simple command-line-based library management system written in Python. It allows users to perform various operations like creating an account, searching for books, adding books, and loaning books.

## Features
- **Create Account**: Users can create a library account by providing their personal details.
- **Search for Books**: Users can search for books in the library using criteria like title, author, publisher, or publication date.
- **Add a Book**: Library staff can add new books to the collection with book details.
- **Borrow a Book**: Registered users can borrow available books.
- **Error Handling**: The system includes error handling to ensure data consistency.
- **User Account Management**: Users can edit their personal details.
- **User Overdue Books**: The system can identify and display overdue books for users.

## Usage
1. Run the script: `python library_system.py`.
2. Choose from the menu options (1/2/3/4/5) to perform operations.
   - To create an account, select option 1 and provide your details.
   - To search for books, select option 2 and choose the criteria and value to search for books. You can also choose to borrow a book.
   - To add a book, select option 3 and provide book details.
   - To borrow a book, select option 4 and enter your username and the title of the book you want to borrow.
   - To exit, select option 5.

## Dependencies
This script does not have external dependencies and can be run using Python 3x.x.
